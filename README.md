# TatvaCare Operations Dashboard

A fully interactive single-file operations dashboard for **CGM Inventory** and **Patient Engagement** tracking.

## 🔗 Live URL
> Auto-generated by Vercel after deployment — e.g. `https://tatvacare-dashboard.vercel.app`

---

## 📦 What's Inside

### CGM Inventory
- **Tatvapractice** & **Goodflip** — switchable via sidebar
- Views: Overview · AM-wise · Area-wise
- Filters: Month dropdown + CGM Type selector (Abbott / Libre 2+ / Tracky TT / Tracky TS)
- Full 4-CGM bifurcation in all charts and tables

### Patient Engagement
- **Inside Sales** & **Field Sales** — switchable via sidebar
- Views: Overview · Monthly Metrics · Cohort Analysis · Coach Assessment
- Metrics: Welcome call booked/completed · Diagnostics · Dr. Consultation · Coach sessions

## 🚀 Deploy on Vercel (first time only — 3 minutes)

1. Go to **[vercel.com](https://vercel.com)** → sign in with GitHub
2. Click **"New Project"** → Import this repository
3. Settings:
   - Framework Preset → **Other**
   - Build Command → *(leave blank)*
   - Output Directory → *(leave blank)*
4. Click **Deploy**

Your live URL is ready in ~30 seconds.

---

## 🔄 How to Update Data

Every time your Excel file changes:

1. Upload the new `.xlsx` to Claude
2. Claude generates a fresh `index.html`
3. Replace `index.html` in this repo:

```bash
# If using Git on your computer:
git add index.html
git commit -m "Update dashboard data - $(date +%Y-%m-%d)"
git push

# Vercel auto-deploys in ~30 seconds after push
```

Or simply drag & drop the new `index.html` directly on GitHub.com into this repository.
